## 📄 Overview

Auto Work Skill is a structured skill workspace for building autonomous agent workflows. It follows a repository-first lifecycle with incremental loading, diagnostics, and isolated execution boundaries. The project is designed to remain lightweight while being easy to extend with additional capabilities.

## 🛠 Core Features

- Git lifecycle automation for workspace synchronization.
- Progressive repository ingestion to minimize unnecessary context.
- Isolated execution sandboxes for safer task handling.
- Structured diagnostics and optimization pipelines.
- Agent-ready repository layout for skills, workflows, and references.

## ⚙️ Installation

1. Clone the repository.
2. Review `SKILL.md` to understand the skill definition.
3. Update `config.json` and workflow files as needed.
4. Integrate the skill into your Claude Code or supported IDE workflow.

## 🚀 Usage

1. Open the repository in your supported agent environment.
2. Load the skill definition from `SKILL.md`.
3. Follow the repository workflow and capability references.
4. Extend the automation by modifying the structured configuration and workflow files.
